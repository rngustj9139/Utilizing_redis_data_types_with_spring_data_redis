# [Redis 특징]
### 인 메모리 key-value DB이며 O(1)의 시간 복잡도를 갖는다.
### 싱글 스레드 기반의 No-SQL DB이기 때문에 동시성 문제가 발생하지 않는다.
### pub/sub 시스템 기반으로 다양한 기능의 구현이 가능하다.
